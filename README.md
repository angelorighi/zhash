zhash
=====

Compresses a directory and calculates the sha1 hash.


###Usage###

<pre>
Zhash v0.3: Zip & Hash a directory
Usage: zhash.py -d <directory> [-v] [-h]
  -d, --directory <directory>: Directory to zip
  -o, --output <filename> (without extension): Output file
  -v, --verbose: Verbosity
  -h, --help: This help
</pre>
